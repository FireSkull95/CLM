Olymp modular

The "Olymp" range of Modular Shingles consists of ready to install elements, which can be used to cover roofs of any shape and size.

Each individual module is a sheet of a set size with special geometry, which allows mounting metal shingles in an easy and seamless fashion, similar to toy bricks. During the installation, metal shingle "steps" (hidden connection points under each module) creates an effect of seamless connections along the length of coverage.

Installation of metal shingles is a fairly easy task. Thanks to its ergonomic size (standard module size is 1.190m x 0.720 m) the amount of waste is minimal, which is one of the biggest advantages of modular singles when compared to other traditional roofing products.
